Motor AceStream

Conecta tu motor local dockerizado con GitHub Pages mediante un túnel HTTPS.

Sin configurar
1
Levanta el motor Docker
En la carpeta del repo clonado
docker compose up -d
2
Crea el túnel temporal
Deja esta terminal abierta mientras ves contenido
cloudflared tunnel --url http://localhost:8080
3
Pega la URL HTTPS del túnel
Algo como https://xxxx-xxxx.trycloudflare.com
ℹ️
Qué valida el estado verde

La app comprobará la versión del motor por HTTPS a través del túnel. Si eso falla, no es un problema visual: es CSP, CORS o redirect del motor.
