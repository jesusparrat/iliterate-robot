🔧 Configurar túnel AceStream

GitHub Pages sirve desde HTTPS. Para reproducir streams ACE de forma nativa necesitas exponer tu motor local con HTTPS usando un túnel.

1
Levanta Docker docker compose up -d
2
Instala cloudflared winget install Cloudflare.cloudflared
3
Crea el túnel temporal cloudflared tunnel --url http://localhost:8080
4
Pega la URL HTTPS que te da cloudflared aquí
