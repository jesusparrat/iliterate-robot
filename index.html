Motor AceStream

Conecta tu motor local dockerizado con GitHub Pages mediante un túnel HTTPS.

Sin configurar
1
Levanta el motor Docker
En la carpeta del repo clonado
docker compose up -d
2
Instala cloudflared solo una vez
winget install Cloudflare.cloudflared
3
Crea el túnel temporal
Deja esta terminal abierta mientras ves contenido
cloudflared tunnel --url http://localhost:8080
4
Pega la URL HTTPS que genera cloudflared
Algo como https://xxxx-banana.trycloudflare.com
ℹ️
¿No tienes el motor configurado?

Clona el repo, asegúrate de tener Docker instalado, y ejecuta docker compose up -d desde la raíz del proyecto. El motor AceStream quedará disponible en http://localhost:8080.
